# ethereumH

This is the build repository for all the components of BlockApps'
Haskell Ethereum client.  It is managed by the ["gitmap"
tool](http://github.com/ryanreich/gitmap).  After installing that, get
started with the following commands:

```
git clone git@github.com:blockapps/ethereumH -b gitmap
cd ethereumH
gitmap clone -b develop
stack install
```
